*Покупка стикеров системой с стороних мркт*
Каждые 1час (время на решение продать(для системы это значит вернуть в pool) или вывести) должна быть авто-покупка чтобы возвращять кейсы в активное состояние
Исключение:
- Кол-во стикеров не больше чем planned_pool(Unique параметр для каждого стикера в каталоге)
- Стикеры не дороже n-ой суммы
- Стикеры не дороже n% чем флор цены на данный момент времени в стикеркаталоге

*Сбор флора/раз в сутки*
Собирается цены на все стикеры из стикеркаталога
Методика сбора
- В зависимости от issuiser,применяются разные api
- Для стикеров с is_onchain = true, применяется api Getgems
- Для стикеров с is_onchain = false, применяется специфичный api для каждого issuier

Исключение:
